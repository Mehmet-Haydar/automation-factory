# DemoRun_MixingStation — live end-to-end demo

> ⚠️ **Synthetic example — no customer data.** A freely-invented 1997 Siemens
> S7-300 mixing station (star-delta stirrer pump + conveyor + cooling valve +
> 4-20 mA temperature). Standard automation patterns only — nothing
> patent/IP-encumbered.

This is the project behind the screenshots in the main [README](../../README.md)
(`docs/img/demo/`). It was produced **live** by the pipeline, end to end:

1. **Input** — `_input/legacy.awl`: synthetic S7-300 STL legacy code (star-delta
   starter, manual M-bit step chain, analog heater with hysteresis, and an
   E-Stop wired to a *standard* output with a maintenance bypass that defeats it).
2. **14-Point Pack** — `metadata/RD01..RD14.md`: generated by the AI extraction
   pipeline (DeepSeek), every RD stamped `DRAFT_UNVERIFIED`.
3. **Safety (RD05)** — the AI correctly flagged the E-Stop-on-standard-PLC and
   the `M50.0` bypass that disables it; SIL/PLr/Category left **blank** for a
   certified safety engineer (human-in-the-loop by design).
4. **Code → TIA** — the library-first SCL (`FB_Motor_DOL`, `FB_Motor_StarDelta`,
   `FB_AnalogScale`, OB_Main, …) was imported into a **real TIA Portal V19**
   project and **compiled with 0 errors** — see `docs/img/demo/`.

`AI_DECISION_LOG.jsonl` is the EU-AI-Act audit trail: every AI step (model,
provider, prompt/output hash) recorded.

## 2. When to use

Mid-range retrofit of an S5/S7-300 line: enough memory and PROFINET for
distributed ET200SP IO, a KTP/Unified HMI and a handful of drives. For
safety add an F-CPU variant (1516F) — this standard CPU has no F-runtime.

## 3. PROFINET Address Model

```
X1 (PN/IE): line/star to ET200SP interface modules (IM155-6 PN).
Device names + IP set per project (default 192.168.0.1 — set in HW config).
```
